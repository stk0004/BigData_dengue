Requisitos
Python 3.x

PySpark instalado (pip install pyspark)

Java JDK (versão 11 ou superior) instalado

Importante
O PySpark depende do Java para funcionar corretamente. Antes de rodar o script, certifique-se de:

Ter o Java JDK instalado (versão 11 ou superior).

Configurar a variável de ambiente JAVA_HOME apontando para o diretório onde o Java está instalado.

Incluir %JAVA_HOME%\bin no PATH do sistema.

Sem o Java corretamente instalado e configurado, o script pode apresentar o erro:
Java not found and JAVA_HOME environment variable is not set.
