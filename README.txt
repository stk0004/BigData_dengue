# Análise de Dados da Dengue com PySpark

Projeto desenvolvido para a disciplina de **Tópicos de Big Data**.

##  Tecnologias Utilizadas
- Apache Spark (PySpark)
- Hadoop (conceitualmente para armazenamento HDFS)
- Pandas / Matplotlib
- Python 3

##  Dataset
Simulação de dados de casos de dengue por ano, estado e classificação (leve ou grave).

##  Arquivos
- `dengue.csv`: Dataset simulado
- `dengue_analysis_spark.py`: Script principal
- `grafico_dengue_ano.png`: Gráfico gerado
- `Apresentacao_Dengue_PySpark.pptx`: Slides da apresentação
